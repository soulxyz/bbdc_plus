# 🚀 快速开始指南

## 📦 安装（仅需一次）

### 方法一：使用安装脚本（推荐）

```bash
双击运行 install.bat
```

### 方法二：手动安装

1. **安装 Python 依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **安装 Tesseract OCR**
   - 下载：https://github.com/UB-Mannheim/tesseract/wiki
   - 安装时勾选 "Add to PATH"

## 🎯 使用步骤

### 1. 启动程序

**推荐方式：**
```bash
右键 run.bat → 以管理员身份运行
```

**或者：**
```bash
python main.py
```

### 2. 选择识别区域

程序启动后会自动弹出选择界面：

1. 打开你的背单词软件
2. 用鼠标拖拽选择显示单词的矩形区域
3. 释放鼠标完成选择

**提示：** 
- 选择区域要尽量准确框住单词
- 背景越简单，识别率越高
- 字体越大，识别越准确

### 3. 开始使用

选择完成后：
- 程序自动开始识别
- 悬浮窗会实时显示单词信息
- 可以拖动悬浮窗调整位置

## ⌨️ 快捷键

| 按键 | 功能 |
|------|------|
| `F2` | 重新选择识别区域 |
| `F3` | 显示/隐藏悬浮窗 |
| `F4` | 暂停/继续识别 |
| `ESC` | 退出程序 |

## 💡 使用技巧

### 配合背单词软件

推荐使用场景：
1. **墨墨背单词** - 选择单词卡片区域
2. **不背单词** - 选择单词显示区域
3. **扇贝单词** - 选择中心单词区域
4. **Anki** - 选择卡片正面

### 优化识别效果

1. **选择合适的区域**
   - 只框住单词本身
   - 避免包含过多其他文字
   - 背景尽量纯色

2. **调整软件设置**
   - 增大单词字体
   - 使用简洁的界面主题
   - 提高屏幕亮度

3. **处理识别错误**
   - 程序内置模糊匹配
   - 会自动纠正常见 OCR 错误
   - 如果经常识别错误，按 F2 重新选择区域

## ❓ 常见问题

### 快捷键不工作？
→ 以管理员权限运行程序

### 识别不到单词？
→ 检查 Tesseract 是否正确安装
→ 尝试重新选择更清晰的区域

### 悬浮窗遮挡内容？
→ 拖动悬浮窗到其他位置
→ 按 F3 临时隐藏

### 找不到单词？
→ 数据库包含 2500+ 常用词
→ 一些生僻词可能没有收录

## 🎓 高级功能

### 自定义配置

编辑 `config.py` 文件可以调整：

- 识别间隔时间
- 窗口透明度和大小
- 字体和颜色
- 快捷键映射

### 调试模式

如需查看详细日志，在 `config.py` 中设置：
```python
DEBUG = True
```
